# Digital Forensics Portfolio
This repository contains ethical, controlled lab case studies simulating real-world digital forensics investigations.

## Overview
Each case demonstrates practical skills in:
- Windows Event Log analysis
- Malware triage
- Acount breach investigation
- Timeline reconstruction
- Security recommendations

## Case List
1. **Suspicious Login Investigation** - Simulated unauthorized login detection and timeline reconstruction.
2. **Malware Triage** - Safe malware file analysis and reporting of indicators of Compromise (IOCs).
3. **Account Breach Investigation** - OSINT-style account investigation and recommendations.

## Disclaimer
All investigations were conducted in controlled lab environments. No real systems or sensitive data were compromised.
